# ATM Interface

ATM Interface is a Java-based banking simulation application that allows users to perform basic ATM operations such as depositing money, withdrawing funds, and checking account balance. The project is developed using Object-Oriented Programming (OOP) concepts to provide a simple and interactive banking experience through a console-based interface.

## Features
- Deposit Money
- Withdraw Money
- Check Account Balance
- Transaction Validation
- User-Friendly Menu System

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Classes and Objects
- Encapsulation
- Scanner Class
- Loops and Switch Statements

## How It Works
1. User selects an option from the ATM menu.
2. The system processes deposit or withdrawal requests.
3. Available balance is updated automatically.
4. Users can view their current account balance at any time.
5. The application continues until the user chooses to exit.
